Techtonic

Round 1: Word-Puzzle

We will provide you word puzzle which has various coding related keywords in it. The team which finds most words in least time will qualify for next round. (Time Limit : 5 minutes.)

Round 2: KBC

Coding Quiz in which logical problems will be asked. There will be multiple-choice questions with a single correct answer. Team scoring the maximum points qualify for next round.

Round 3: Design-Pattern

Each team will be given problem statement whose output is a particular pattern. Team has to design the pattern in minimum possible time. Team must be able to explain their logic. (Time Limit : 3 hours)

Round 1: Jumbled-Code

You will be provided code which has all the lines jumbled. Output of the code will be provided. Your task is to arrange code as per the output required. (Time Limit: 25 min.)

Round 2: Debugging

Code having syntax, semantic, compile time or runtime errors will be given to participants in this round. You will have to debug the errors to make the code work fine. Team scoring the maximum points by debugging in minimum time qualify.

Round 3: Code-Relay

Each team will be given problem statement whose output can be obtained by two distinct logical approaches. Team will be given 10 minutes to discuss the problem statement. In 2 hours, each team has to get output in both ways. One participant will start coding while other will wait until he's/she's done.
